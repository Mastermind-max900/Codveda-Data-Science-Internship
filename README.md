Task 1: Data Collection and Web Scraping
Codveda Technologies - Data Science Internship
Project Overview
This project demonstrates the collection of data from a website using automated web scraping techniques. The goal was to build a structured dataset (CSV) that mirrors the format of a sentiment analysis dataset, capturing text, authors, and associated tags.
Objectives Met
Target Identification: Inspected the HTML structure of Quotes to Scrape to identify relevant data tags.
Technical Implementation: Developed a Python script utilizing the BeautifulSoup and requests libraries to extract live data.
Data Storage: Organized the scraped content into a structured .csv format using pandas for easy analysis.
Problem Solving: Successfully handled initial challenges including environment setup (ModuleNotFoundError) and HTML tag mismatches (AttributeError).
Tools Used
Language: Python
Libraries: requests, BeautifulSoup4, pandas
Environment: VS Code on HP ZBook G8
How to Run
1. Clone this repository.
2. Install dependencies: pip install requests beautifulsoup4 pandas.
3. Run the script: python tas1_scraper.py.
